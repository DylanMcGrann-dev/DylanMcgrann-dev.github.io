Projects

Budget Notebook

Budget Notebook is a website designed to help people with creating and sticking to a budget. Using React.js users are able to create, add and update budgets with live updates. Savings goals can be set on the budget page as well as viewing savings goals progress compared to previous months.

On the transactions page users can track their spending by adding, removing or editing transactions. Users can view their spending vs their savings based on month with the line chart at the top of the page.

link to Git repo.

Buy Sell

Buy Sell is an e-commerce marketplace for anyone looking to buy or sell athletic clothing. Users are able to browse clothing by gender, category or all. Once a user has created an account they gain access to the abilities of favoriting items, contacting sellers and posting items to sell.

link to Git Repo.
